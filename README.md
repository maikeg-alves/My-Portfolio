
# Meu Portfólio

### 🚧⚠️ PROJETO EM MANUTENÇÃO ⚠️🚧 

Depois de muito estudo e uma boa carga de conhecimento decidir por criar meu 
portfólio e mostrar par o mundo meu trabalho e meus conhecimentos na área, acredito que esse 
Portfólio é minha entrada para o mercado de trabalho.

## Sobre o projeto 

As tecnologias usadas nesse projeto foi React js, Bootstrap, Material-UI (estou fazendo a transição
de Bootstrap para Material-UI), styled components.

já em questão de consumo de dados, o portfólio conta com uma api pessoal minha, escrita em Nodejs e usando o ORM Prisma e para complementar
rodando também a api do GitHub, responsável por pegar dados dos repositórios como
link da repo, data de criação da repo, ultima atualização da repo e a descrição do projeto.

já no envio de email usei nodemailer, uma biblioteca simples de envio de email escrita em Nodejs também.


## Demonstração

Projeto hospedando no heroku.
